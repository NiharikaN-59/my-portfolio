My project name was Intelligent Medicine Dispenser. It is also called as Automated Medicine Dispensing Using Voice Recognition. This project was done based on the IOT.

Intelligent Medicine Dispenser:-

To create a voice-controlled system that helps individuals automatically receive medication based on their symptoms (fever, cough, etc.), using speech recognition and automated dispensing based on the inputted time. An Arduino-based medicine dispenser is triggered to release the required dose, and emergency notifications are sent to healthcare professionals if needed. The system enhances medication adherence and provides a safer, more user-friendly experience for patients, especially those with chronic illnesses or the elderly.

Key Features:
Voice Recognition: Captures symptoms from the user.
Medicine Dispensing: Controls dispensing mechanisms (e.g., Arduino).
Text-to-Speech Feedback: Confirms the action taken.
